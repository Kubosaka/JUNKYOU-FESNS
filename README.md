# 環境構築
## MACの場合
### nodeのインストール
```
brew install nodebrew
```
nodebrewがインストールされているかの確認
```
nodebrew ls
```
nodeのインストール
```
brew install node
```
nodeが入っているかの確認
```
node -v
```
nodeがインストールされると自動的にnpmとnpxが入るようになる
npmが入っているかの確認
```
npm -v
```
npxが入っているかの確認
```
npx -v
```
### リポジトリのクローン
自分の作業ディレクトリに入って以下を実行
```
git clone 
```
残りはあとでやります
